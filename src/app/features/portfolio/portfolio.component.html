• K. Lyricsopen_in_new Angular AWS Serverless

    A front-end to present the AWS Genius Lyrics service. The featured artist in this demo is rapper, Young Thug. This web app is written with Angular. It is deployed with AWS S3 and made available with a CDN using AWS Cloudfront. The domain is managed using Amazon Route 53 and the cert is provisioned with Certificate Manager.
